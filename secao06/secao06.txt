39. O que são Hooks?

Funções que permitem que sejam usados os States e outros recursos do React sem que seja
necessário escrever uma classe, ou seja, é possível usar todos os benefícios de uma classe diretamente
em uma função.

Síntaxe

useNomeDoHook()

useState()


Hooks são funções que permitem a você "ligar-se" aos recursos de state e cliclo de vida
do React a partir de componentes funcionais.

Hooks não funcionam dentro de classes, eles permitem que você use React sem classes.


Hooks são completamente opcionais. Você pode experimentar Hooks em alguns componentes
sem reescrever nenhum código existente.


Hooks são 100 retrocompatíveis. Hooks não possuem nenhuma quebra de compatibilidade.
