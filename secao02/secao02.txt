3. O que é React?

O React é uma biblioteca JavaScript declarativa, eficiente e flexível para criar interfaces com o usuário.

É possível criar componentes que podem ser utilizados em uma aplicação Web.



O que é DOM?

Quando uma página da Web é carregada, o navegador cria uma hierarquia dos objetos chamada de Modelo de Documento por Objetos.



Como o React funciona?

Os componentes ficam interligados, em uma espécie de hierarquia, "árvore" e cada componente têm sua função.

Esta estrutura criada pelo React é chamada de DOM virtual, onde são feitas todas as alterações necessárias antes de 
alterar o DOM real do navegador.

Uma vantagem do React é que ele muda apenas o que precisa ser mudado.

O que é um Componente?

Componentes permitem que você divida a UI em partes independentes, reutilizáveis e pense em cada parte isoladamente.


function BemVindo(props){
	return <p>Olá, {props.nome}</p>;
}


class BemVindo extends React.Component{
	render(){
		return <h1>Hello, {this.props.nome}</h1>;
	}
}
